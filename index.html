Assignent 1

The first group assignment was to visualize some set of information. Our group (named Gangnam Style) created a circular representation of the courses included in our study program at Information Networks, Aalto University.

On the technical side, the visualization is made with Processing, using a plugin to facilitate development in the Eclipse IDE.

We experienced issues with getting the applet to work, so at the moment the visualization can only be run as a standalone program on Mac and Windows, respectively.

The code for the project can be viewed at the project GitHub page.
